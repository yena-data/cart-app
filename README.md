# 장바구니 앱입니다.

어머니께 선물 하세요

1. 장 볼 오늘의 물건을 입력하고 추가 할 수 있습니다.
   - 아무것도 입력하지 않으면 안됩니다.
   - 장 볼 물건을 추가하시면, 새로고침해도 없어지지 않습니다.
2. 장 볼 물건을 삭제 할 수 있습니다.
   - **주의사항** 한번 삭제 시 복구 불가능
   - 새로고침해도 안됩니다.
3. 장 볼 물건을 수정 할 수 있습니다.
   - 수정 할 때에는 이미 장보기 목록에 있는 이름으로 수정 할 수 없습니다.
